# Конфіги для звірки

Три файли потрібно заповнити **до першого запуску**:

| Файл | Що в ньому |
|------|------------|
| `pidrozdily.xlsx` | Список підрозділів (Instagram-сторінок) + до якої картки Privat прив'язана кожна |
| `oborot-pidrozdil.xlsx` | Оборот (виручка) кожного підрозділу помісячно — для пропорційного розподілу витрат |
| `pravyla-vytrat.xlsx` | Як розподіляти затрати: статті, типи (`пропорційно` / `прямо`), ключові слова для розпізнавання |

## Швидкий старт

Якщо вже встановлений Python і залежності — просто:

```bash
python -m recon init-config
```

Створить шаблонні `.xlsx` з прикладами. Відкривай в Excel і заповнюй.

Якщо ще не ставив залежності — можеш руками створити Excel-файли з такими колонками:

### pidrozdily.xlsx

| код | назва_1С | назва_Instagram | картка_Privat |
|-----|----------|-----------------|---------------|
| 1   | Магазин_А | @magazyn_a     | 4***1234      |
| 2   | Магазин_Б | @magazyn_b     | 4***5678      |

- `назва_1С` — точна назва підрозділу як вона у тебе в довіднику 1С.
- `картка_Privat` — частина номеру картки на яку клієнти платять. Може бути замаскована, програма порівнює за останніми 4 цифрами. Якщо одна спільна картка — лиши порожньо.

### oborot-pidrozdil.xlsx

| рік | місяць | підрозділ | оборот |
|-----|--------|-----------|--------|
| 2024 | 1     | Магазин_А | 50000 |
| 2024 | 1     | Магазин_Б | 30000 |
| 2024 | 2     | Магазин_А | 60000 |
| ...  |        |           |        |

- `оборот` — виручка/реалізація підрозділу за місяць у гривнях.
- Можна вивантажити з 1С звітом «Продажі по підрозділах».
- **Один рядок = підрозділ × місяць.**

### pravyla-vytrat.xlsx

| стаття_витрат | тип_розподілу | прямий_підрозділ | ключові_слова |
|---------------|---------------|------------------|---------------|
| Оренда         | пропорційно   |                  | "оренда","орендна" |
| Реклама ІГ Магазин_А | прямо    | Магазин_А        | "facebook","fb.com" |
| Комунальні     | пропорційно   |                  | "ддсу","водоканал","газ" |
| Постачальник Х | прямо         |                  | "тов постачальник х" |

- `тип_розподілу = пропорційно` → розподіляємо суму на всі підрозділи згідно з оборотом.
- `тип_розподілу = прямо` → вся сума на `прямий_підрозділ`. Якщо порожньо — алгоритм спитає в звіті.
- `ключові_слова` — список через кому в лапках. Програма шукає їх у призначенні платежу / назві контрагента.
  - Регістр не важливий.
  - Якщо знайшла ≥1 ключове слово — застосовує це правило.
  - Якщо нічого не співпало — у звіті ставить `?` біля статті.

## Як онбордити нові правила

Коли у звіті бачиш багато рядків з `?` у статті — додай нове правило в `pravyla-vytrat.xlsx` і запусти `recon` ще раз. Програма сама перерозподілить.
