# Обробка 1С для масової проводки (BankReconciliation.epf)

Цей файл — інструкція як створити обробку в Конфігураторі 1С, яка читатиме
`output/import-YYYY-MM.xlsx` і створюватиме ПКО/ВКО у касі.

> **Статус:** .epf поки **не створено**. Цей файл — план і скелет коду.
> Я підготую повний текст модуля коли ти запустиш Конфігуратор і скажеш точну
> версію УТП/УНФ (видно у вікні «Про програму»).

## Що робитиме обробка

1. **Кнопка «Вибрати файл»** → діалог вибору `import-YYYY-MM.xlsx`.
2. **Таблиця прев'ю** на формі — показує всі рядки з файлу: дата, тип, сума,
   контрагент, підрозділ або розподіл.
3. **Кнопка «Створити документи»** — обробляє таблицю:
   - Рядок «До проведення» з типом **ПКО**:
     - Шукає або створює Контрагента (за ЄДРПОУ або ПІБ).
     - Якщо вказана Реалізація-основа — знаходить її в базі.
     - Створює ПКО (Прибутковий касовий ордер), заповнює реквізити, проводить.
   - Рядок «До проведення» з типом **ВКО**:
     - Якщо розподіл = `прямо` — один ВКО на вказаний підрозділ.
     - Якщо розподіл = `пропорційно` — **окремий ВКО на кожен підрозділ** з відповідною часткою.
   - Рядок «Пересорт»:
     - Знаходить існуючий ПКО за номером + датою + сумою.
     - Скасовує проведення, змінює підрозділ, проводить заново.
4. **Кнопка «Скасувати останній імпорт»** — видаляє всі документи зі службовим
   коментарем `[recon-YYYY-MM]`. Аварійний відкат.

## Як я створю обробку (коли почнемо)

1. Ти відкриваєш Конфігуратор 1С на робочій базі.
2. **Файл → Новий → Зовнішня обробка**.
3. Створюєш форму обробки (Форми → Додати).
4. На формі:
   - реквізит `ШляхДоФайлу` (рядок, 500),
   - реквізит `Таблиця` (з колонками: Дата, Тип, Сума, Контрагент, Підрозділ, Стаття, Основа, Розподіл, Статус),
   - кнопки `Вибрати`, `Створити`, `Скасуватиімпорт`, `Очистити`.
5. У модуль форми вставляєш текст який я підготую (≈300-500 рядків 1С-коду).
6. Зберігаєш як `BankReconciliation.epf`.
7. Закриваєш Конфігуратор, відкриваєш режим Підприємство.
8. **Файл → Відкрити → BankReconciliation.epf**.

## Скелет модуля (для прикидки обсягу)

```bsl
&НаКлієнті
Процедура ВибратиФайлНаКлієнті(Команда)
    Діалог = Новий ДіалогВибораФайлу(РежимДіалогуВибораФайлу.Відкриття);
    Діалог.Фільтр = "Excel|*.xlsx";
    Якщо Діалог.Вибрати() Тоді
        ШляхДоФайлу = Діалог.ПовнеІм'яФайлу;
        ЗавантажитиТаблицюНаСервері();
    КінецьЯкщо;
КінецьПроцедури

&НаСервері
Процедура ЗавантажитиТаблицюНаСервері()
    // Відкриваємо xlsx через COM Excel.Application.
    Excel = Новий COMОб'єкт("Excel.Application");
    Книга = Excel.Workbooks.Open(ШляхДоФайлу);
    // ... читання листа "ДоПроведення" ...
    Excel.Quit();
КінецьПроцедури

&НаСервері
Процедура СтворитиДокументиНаСервері()
    Для Кожного Рядок Із Таблиця Цикл
        Якщо Рядок.Тип = "ПКО" Тоді
            СтворитиПКО(Рядок);
        ІнакшеЯкщо Рядок.Тип = "ВКО" Тоді
            СтворитиВКОзРозподілом(Рядок);
        КінецьЯкщо;
    КінецьЦикл;
КінецьПроцедури

&НаСервері
Процедура СтворитиПКО(Рядок)
    Док = Документи.ПрибутковийКасовийОрдер.СтворитиДокумент();
    Док.Дата = Рядок.Дата;
    Док.СумаДокумента = Рядок.Сума;
    Док.Контрагент = ЗнайтиАбоСтворитиКонтрагента(Рядок.Контрагент, Рядок.ЄДРПОУ);
    Док.Підрозділ = Довідники.ПідрозділиОрганізацій.ЗнайтиЗаНайменуванням(Рядок.Підрозділ);
    // Якщо є основа — Реалізація:
    Якщо ЗначенняЗаповнено(Рядок.Основа) Тоді
        Док.ДокументОснова = ЗнайтиРеалізацію(Рядок.Основа);
    КінецьЯкщо;
    Док.Коментар = "[recon-" + ПеріодЗвіту + "] Створено автоматично звіркою Privat";
    Док.Записати(РежимЗапису.Проведення);
КінецьПроцедури

// ... аналогічно ВКО з циклом по підрозділах розподілу ...
```

> Це **псевдокод** — точні імена реквізитів і документів залежать від конфігурації
> (УТП 1.3 і УНФ 1.6 мають різні структури). Реальний модуль я підготую коли побачу
> метадані твоєї бази.

## Що потрібно від тебе перед стартом фази 3

1. Відкрий 1С в режимі Підприємство → Довідка → Про програму. Зроби скрін.
   Там написано назву конфігурації і реліз (наприклад «Управління торговим
   підприємством для України, редакція 1.2.69»).
2. Відкрий Конфігуратор → Дерево метаданих → Документи → знайди «Прибутковий
   касовий ордер» (або «ПКО»). Розкрий і скажи назви реквізитів:
   - Як називається реквізит «Контрагент» (може бути `Контрагент`, `Покупець`, `Платник`)
   - Як називається реквізит «Підрозділ» (може бути `Підрозділ`, `ПідрозділОрганізації`, `СтруктурнийПідрозділ`)
   - Є реквізит «ДокументОснова» чи треба заповнювати таблицю «Розшифровка платежу»?
3. Те саме для «Видатковий касовий ордер».
4. Назва довідника підрозділів (`Підрозділи` / `ПідрозділиОрганізацій` / `СтруктурніПідрозділи`).

Це 5-10 хвилин у Конфігураторі. Після цього я напишу повний модуль за пів дня.

## Безпека

- Перший прогін **обов'язково на копії бази**! Створи копію через
  Конфігуратор → Адміністрування → Завантаження/Розвантаження інформаційної бази.
- Кожен документ створюється з коментарем `[recon-YYYY-MM]` — за цим легко
  знайти й відкотити.
- Поки .epf не відлажена — обмежся 10-20 рядками за один прогін.
